# Cyranth GitHub Pages

This repository contains the static GitHub Pages landing page for Cyranth.

No build step is required. The site uses plain HTML, CSS, JavaScript, SVG, `robots.txt`, and `sitemap.xml`.

## Local Preview

- VS Code Live Server
- `py -m http.server 8080`

## Deployment

Push to the `main` branch. GitHub Pages serves the site from the repository.

## Privacy

No real name, phone number, personal email, or private project data is used.
